Installation
============

Package can be installed from PyPi:

.. code:: bash

   pip install scikit-na

or from this repo:

.. code:: bash

   pip install git+https://github.com/maximtrp/scikit-na.git

Requirements
~~~~~~~~~~~~

* NumPy
* Statsmodels
* Seaborn
* Pandas
* Altair
* Matplotlib
* ipywidgets
